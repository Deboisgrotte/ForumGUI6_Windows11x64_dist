# Installation Instructions

Follow these steps to install and run the application:

## 1. Download and Extract
- Download the provided ZIP file containing the application.
- Unzip the contents to a folder in your desired location on your computer.

## 2. Folder Structure
- Ensure the entire unzipped folder (including all subfolders such as `_internal` and `file_storage`) remains intact.
- **Do not move the executable file out of this folder.** The executable must stay in the same folder as the `_internal` and `file_storage` directories to function correctly.

## 3. First Launch (Windows Security Warning)
- The first time you open the application, Windows may display a security message: **"Windows protected your PC"**.
- Click **More info**.
- Then click **Run anyway** to start the application.

## 4. Running the Application
- Double-click the executable file to launch the app.
- The app will now have access to all required internal files and storage.

---

**Note:**
- Do not delete or move any files or folders inside the application directory.
- If you encounter issues, ensure all files are present and the folder structure is unchanged.
